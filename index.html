What is Docs over SSH?

Docs over SSH is an emerging pattern where documentation is exposed as a navigable filesystem over SSH instead of a traditional web interface.

It is especially useful for AI coding agents and CLI-first workflows, where tools need structured, scriptable access to documentation.

Instead of browsing pages, you explore documentation using terminal commands like ls, cat, and grep.
