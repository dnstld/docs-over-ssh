Documentation over SSH

A new way to explore developer documentation — directly from your terminal.

ssh supabase.com

What is this?

Some platforms are starting to expose their documentation through SSH. Instead of opening a browser, you connect via your terminal and navigate docs interactively — like a terminal UI.

First real implementation

Supabase

The first known platform to offer a full documentation experience over SSH.

ssh supabase.com
Read announcement →

Why this is interesting

  • Works natively in the terminal
  • No browser required
  • Interactive and keyboard-driven
  • Feels natural for developers
  • Opens a new interface for documentation

Build your own

This pattern is just starting. You can build your own documentation over SSH using tools like:

  • SSH servers (Node.js, Go)
  • Terminal UI frameworks
  • Libraries like Wish (Go)

This is an open space — there are no standards yet.
